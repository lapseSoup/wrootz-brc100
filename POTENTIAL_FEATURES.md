# Potential Features

Ideas for future development.

## Search & Discovery
- **Archive search toggle** - Option to search expired tags exclusively, useful for finding historical content that no longer has active wrootz
- **Relevance-weighted search** - Weight matches differently (title > body > active tag > expired tag) instead of pure wrootz ranking
- **Related tags** - When viewing search results for a tag, show a list of top related tags (tags that frequently appear together)

## Feed & Filtering
- **Sort options** - Allow sorting by newest, most wrootz, trending (recent gains)
- **Tag filtering in sidebar** - Click a tag to filter feed by that tag

- **Post categories** - Optional categorization beyond tags
- **Comments on posts** - Allow users to comment on posts with threaded discussions

## Analytics
- **Personal dashboard** - Show user's total wrootz earned, lock history, profit from sales
- **Tag analytics** - Show tag performance over time

## Monetization
- **Tipping** - Direct BSV tips to creators/owners separate from locking
- **Premium features** - Analytics, custom profiles, etc.
